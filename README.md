# Student AI Assistant

## Overview
Student AI Assistant is an AI chatbot developed for IBM SkillsBuild using IBM watsonx Orchestrate concepts.

## Features
- Programming Help
- AI Guidance
- Career Guidance
- Student Support

## Technologies
- Python
- Streamlit
- IBM watsonx Orchestrate
- IBM Cloud
